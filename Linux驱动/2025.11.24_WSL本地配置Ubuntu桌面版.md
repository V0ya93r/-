#  ==重点==
确保可以直接使用Windows资源管理器进行Ubuntu的文件操作，要把默认用户设置为root
如果默认为voyager,使用命令
```
sudo -s
cd /etc
vim wsl.conf
```
添加
```
[user]
default=root
```
之后再cmd里使用
```
wsl --shutdown
```
重启
